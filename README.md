# 智慧盲打訓練系統 (Smart Typing Training System)

## Project Overview
本專案為一套結合電腦視覺與互動回饋的智慧盲打訓練系統，
透過即時輸入分析與視覺回饋，協助使用者建立正確盲打習慣。

---

## Motivation
原先專案方向為 3D 重建與影像分割前處理，但因開發時程與系統整合考量，
調整為以即時互動與實用性為主的智慧盲打訓練系統。

---

## Features
- 即時鍵盤輸入/指法偵測
- 錯誤提示與回饋
- 訓練模式（遊戲化）
- OpenCV 即時影像處理

---

## Tech Stack
- Python
- OpenCV
- NumPy
- Mediapipe（若有使用請保留）
